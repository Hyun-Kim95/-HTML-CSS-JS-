JavaScript

JavaScript dgavo skript often abbreviated as JS is a high level dynamic weakly typed prototype- based multi paradigm and interpreted programming language Alongside HTML and CSS JavaScript is one of the three core technologies of World Wide Web content production It is used to make webpages interactive and provide online programs including Video games The majority of websites employ it and all modern web browsers support it without the need for plug ins by means of a built in JavaScript engine Each of the many JavaScript engines representa different implementation of JavaScript all based on the ECMAScript specification with some engines not supporting the speo fully and with many cngines supporting additional features bevond ECMA

'JavaScript'는 동적인 언어
'script'테그를 쓰면 자바스크립트로 인식
'document.write()'글을 써주는데 1+1과 같이 적으면 계산된 값이 출력
'onclick'뒤에는 무조건 자바스크립트가 와야함
'input'
-'type="button"' 버튼을 만듬
-'value=""' 버튼에 글을 씀
-'onclick="alert()"' 클릭했을 때 경고창에 띄울 글
-'onclick="document.querySelector('')"' 선택자(클릭시 가져올 태그(하나))
-'querySelectorAll('')' 태그선택(전부)
-'type="text"' 글쓰는 칸을 만듬
-'onchange="alert()"' 글의 내용이 바뀌면 실행될 자바스크립트코드(alert)
-'onkeydown="alert()"' 칸 위에서 아무 키를 누르면 실행될 자바스크립트코드(alert)
-웹사이트-검사-console 들어가면 보고있는 웹사이트에 대해서 자바스크립트를 실행할 수 있다.
ex) alert("".length) 로 문장의 길이를 확인 가능(alert은 안써도 됨)
-"".toUpperCase() 문자들을 대문자로 바꿈
-"".indexof('') 찾는 문자열의 위치 반환(왼쪽(0)부터)
-"".trim() 공백제거
변수를 선언할 때는 앞에 'var'을 붙혀주는게 좋음
- 변수로 중복을 제거
'if () {} else {}' 조건문
-'===' 같다
-'this' 자기 자신을 참조
'var 변수 = [];' 배열(array) 생성
'document.write(변수이름[0]);' 배열의 0번째 값 꺼내기
'document.write(변수이름.length);' 배열의 길이
'변수이름.push('');' 내용물 추가
'while(true or false){}' 반복문
-'var i = 0;' 의 형태를 위에 먼저 쓰고 1씩 추가해서 반복횟수를 정하는게 보통
-'+변수이름[i]+'의 형태로 반복문과 배열을 합쳐서 사용 가능
'console.log()'사이트 검사에서 검사창에 내용 표시
