HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications With Cascading Style Sheets CSS and JavaScript it forms a triad of cornerstone technologies for the World Wide Web Web browsers receive HTML documents

'ol'은 리스트앞에 숫자를 붙혀줌
'ul'은 리스트 끼리의 구분만 할때 사용
'li'는 리스트라는 의미
'h1~6'은 재목을 사용할 때
'p'는 문단을 구분할 때
'br'은 줄바꿈
'head'는 사이트의 설명을 감쌈
'body'는 사이트의 내용을 감쌈
'title'은 사이트의 제목을 바꿈
'html'은 시작과 끝은 감쌈
'!doctype html'은 시작부분에 사용
'meta charset="utf-8"'은 'utf-8'로 글자를 읽어달라는 의미
'img src'는 이미지 불러오기
'a href'는 링크를 걸어줄 부분과 주소를 입력
-'target="_blank"'새창에서 열기
-'title'은 링크에 대한 설명
'font color=""'글씨 색깔을 바꿈(지금은 안씀)
'!--','--'는 이 사이를 없는 셈 치라는 기호
